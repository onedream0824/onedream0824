## Overview

Built a travel booking management system inspired by large OTA workflows, supporting hotels, flights, rental cars, and travel package operations.

## Key Features

- **Multi-vertical booking management** across major travel categories
- **Discount and deal workflows** for campaign-style pricing
- **Flexible payment options** and checkout handling
- **Global-ready architecture** with high-volume search and reservation flows

## Technologies Used

- **Modern web stack**
- **Travel service integrations**
- **Payment and transaction orchestration**
- **Scalable backend APIs**

## Outcome

Delivered a robust booking platform experience focused on operational scalability, pricing flexibility, and customer convenience.

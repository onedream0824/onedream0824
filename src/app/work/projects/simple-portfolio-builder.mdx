## Overview

Built an intelligent personal finance mobile platform that helps users track expenses, manage budgets, and improve long-term financial habits with actionable AI insights.

## Key Features

- **Smart budgeting**: Real-time budget monitoring with spending breakdowns and alerts.
- **Expense analytics**: Behavioral analysis of spending patterns to highlight overspending and optimization opportunities.
- **Net worth tracking**: Clear portfolio-style visibility into financial position over time.
- **Goal-based planning**: Personalized recommendations tied to savings and spending goals.
- **Secure integrations**: Safe synchronization with connected financial data sources.

## Technologies Used

- **React Native**
- **Node.js**
- **AI analytics and recommendation pipelines**

## Outcome

Delivered a finance assistant experience that transforms raw transaction data into practical daily guidance, helping users make better financial decisions with confidence.

## Overview

Led development of a React Native iOS/Android healthcare networking app focused on scalable architecture and real user engagement. The product combines social networking, professional event discovery, and AI-assisted personalization to help users discover relevant people and sessions faster.

## Key Features

- **Cross-platform architecture**: Built maintainable React Native modules for core user journeys across iOS and Android.
- **AI-powered recommendations**: Integrated OpenAI-powered logic for personalized content and contact discovery.
- **Event workflows**: Delivered seat booking and a venue-style seat planner for event participation.
- **Payments and auth**: Integrated Stripe checkout plus Apple/Google social authentication.

## Technologies Used

- **React Native** for mobile app development
- **OpenAI APIs** for recommendation and discovery experiences
- **Stripe** for secure in-app payments
- **Apple/Google auth** for social sign-in
- **REST API integration** for backend connectivity

## Challenges and Learnings

The main challenge was balancing feature depth with smooth app performance on both platforms. This was solved with careful state management, optimized API usage, and a modular codebase that made it easier to scale new features without regressions.

## Outcome

The app shipped with a solid foundation for healthcare networking, including AI discovery, social identity flows, and monetized event participation-ready for continued growth with minimal technical debt.

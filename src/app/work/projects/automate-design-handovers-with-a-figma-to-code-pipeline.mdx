## Overview

Developed a full-featured Android social platform in Kotlin with video calls, messaging, reels, and post sharing. The project focused on building real-time communication and media-heavy interactions that stay responsive at scale.

## Key Features

- **Social features**: Implemented posts, reels, and interactive messaging workflows.
- **Video calling**: Built real-time video communication with stable session handling.
- **AI moderation**: Added machine-learning modules for nude detection and media safety checks.
- **Cloud backend**: Delivered FastAPI services with AWS integrations for streaming, storage, and real-time communication.

## Technologies Used

- **Kotlin** for Android app development
- **FastAPI** for backend APIs
- **AWS** for cloud infrastructure and media workflows
- **Machine Learning** for automated content moderation

## Challenges and Learnings

The most demanding part was keeping video, chat, and feed experiences fast under heavy concurrent usage. Careful API design, efficient media handling, and performance-focused architecture decisions helped maintain smooth UX.

## Outcome

Shipped a scalable Android social app architecture that supports real-time interactions, secure media handling, and AI-assisted safety controls-ready for iterative feature expansion.

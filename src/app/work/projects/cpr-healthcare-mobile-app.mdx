## Overview

Developed a healthcare-domain mobile app focused on patient/provider communication, secure identity workflows, and location-aware features.

- **User authentication and profiles**
- **Real-time chat workflows**
- **Face ID integration**
- **GPS and map-based utilities**
- **Timer and video functionality**
- **Admin panel support**

## Technologies Used

- **React Native**
- **Laravel**
- **MySQL**
- **Google Maps / GPS**
- **Face Detection**

## Outcome

Shipped a practical healthcare mobile solution with secure access controls and communication features tailored for day-to-day operations.

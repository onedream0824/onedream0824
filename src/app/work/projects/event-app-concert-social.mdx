## Overview

Built a mobile event experience for concert lovers to discover upcoming shows, track attended concerts, and view friend activity in one place.

## Key Features

- **Concert and artist discovery**
- **Social feed for friend activity**
- **Event tracking and attendance history**
- **Cross-platform mobile delivery**

## Technologies Used

- **React Native**
- **Expo**
- **Node.js**
- **Mobile API integration**

## Outcome

Delivered a social event platform that combines discovery and community interactions into a single, easy-to-use mobile app.

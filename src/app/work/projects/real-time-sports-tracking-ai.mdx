## Overview

Developed a real-time sports analytics system to replace manual camera/operator decisions by automatically identifying the most relevant game area in hockey, football, and basketball broadcasts.

## Key Features

- **Automated ROI selection** for key match activity zones
- **Real-time player position extraction** from continuous video input
- **Multi-sport support** with adaptable tracking logic
- **Low-latency analytics pipeline** for live scenarios

## Technologies Used

- **C/C++**
- **Computer Vision**
- **Machine Learning**
- **ROI control algorithms**

## Outcome

Built a robust analytics foundation for live sports tracking, enabling automated insights without constant human operator control.
